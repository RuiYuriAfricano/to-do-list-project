# to-do-list-project

Projeto de Gestão de tarefas

# Funcionalidades

- Adicionar Tarefa
- Divisão por categoria
- Editar Tarefa
- Eliminar Tarefa
- Marcar Tarefa Como concluída

# Stack

- React.Js
- Typescript
- LocalStorage (API)
- React Hooks
